# Feeder

An app to read the RSS Feeds.

## Use cases

1. **User** should be able to **Subscribe** to any **RSS Feed** using the URL of the rss feed file.
2. User should be able to read the **Content** from the RSS Feed.
3. User should be able to mark any content as **READ / UNREAD**.
4. User should be able to configure the amount of time after which read/unread content is deleted from the system.
5. The system should automatically refresh the **Feed Itesm** for all the feeds at some specified (pre-communicated to the users) time.
